Spring Boot Gym REST

Powered by Spring Boot, this gym database project functions as a RESTful web service dedicated to efficient gym management. Comprising three essential tables, it securely manages user details, class reservations, and news content, utilizing unique identifiers like 'Username' in the Users Table, 'Reservation ID' in Reservations Table, and 'News ID' in the News Table, alongside pertinent details for each category.

  • SpringBoot
